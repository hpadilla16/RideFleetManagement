# Ride Fleet Service Proposal

## Founding Client Pricing Proposal

Fecha: 2026-03-26

Cliente: Dealership Loaner Program Prospect

## Executive Summary

Ride Fleet proposes a founding client agreement for a dealership operation that
plans to manage up to `200` loaner vehicles inside the platform.

This proposal is designed to:

- give the client preferential early pricing
- keep the platform competitive against legacy low-cost tools
- remain meaningfully below premium enterprise alternatives
- preserve enough value for onboarding, support, and product growth

The proposal includes:

- one-time onboarding and setup
- Ride Fleet core platform access
- Loaner Program module
- optional add-ons for:
  - Customer Service
  - Puerto Rico Tolls

## Proposed Commercial Structure

### One-Time Setup Fee

`$1,500`

Includes:

- tenant creation and configuration
- location and basic settings setup
- user and role setup
- module activation
- agreement and operational form configuration
- fleet import templates
- go-live configuration support
- admin and ops onboarding

## Monthly Subscription

### Ride Fleet Core + Loaner Program

`$1,250 / month`

Includes:

- Ride Fleet core platform
- Dealership Loaner Program
- up to `200` active loaner vehicles
- reservation workflow
- customer records
- employee operations workflow
- check-out and check-in
- inspection support
- payment workflow
- operational dashboards
- reporting
- standard support

This pricing is intentionally positioned as founding client pricing and should
be described as preferred early-adopter pricing.

## Optional Add-Ons

### Customer Service Module

`$250 / month`

Includes:

- Issue Center
- support case creation
- dispute handling
- case status workflow
- request-for-information workflow
- case history and internal follow-up

### Puerto Rico Tolls Module

`$300 / month`

Includes:

- AutoExpreso provider setup
- health check
- live sync
- manual and bulk import fallback
- automatic toll matching against:
  - tenant fleet
  - assigned vehicle
  - reservation pickup and return time window
- toll review queue
- reservation toll posting
- toll dispute linkage into Issue Center

## Recommended Package Options

### Option A

Core + Loaner only

- Setup Fee: `$1,500`
- Monthly Subscription: `$1,250`

### Option B

Core + Loaner + Customer Service

- Setup Fee: `$1,500`
- Monthly Subscription: `$1,500`

### Option C

Core + Loaner + Customer Service + Tolls

- Setup Fee: `$1,500`
- Monthly Subscription: `$1,800`

## What The Client Receives

### 1. Core Operational Control

The client receives one connected system for:

- reservations
- vehicle assignment
- customer records
- agreement flow
- inspections
- payment workflow
- reporting

### 2. Loaner Program Control

The client receives the dealership loaner workflow with support for:

- intake and borrower flow
- delivery and return tracking
- lane visibility
- shift board visibility
- billing blockers
- closeout support
- reservation-based vehicle workflow

### 3. Modern Admin Control

The client receives:

- user and role access
- tenant-based configuration
- payment gateway flexibility
- module access control
- workflow visibility across admin, ops, and service teams

### 4. Optional Support And Recovery Layers

With add-ons, the client can also activate:

- Customer Service for structured support and dispute workflow
- Tolls for Puerto Rico AutoExpreso sync and reconciliation

## Market Positioning

Ride Fleet is being positioned to sit in a strong middle ground:

- more modern and flexible than legacy low-cost tools
- more transparent than fully custom quotes
- materially more affordable than premium enterprise platforms

Public market references support this positioning:

- ARSLoaner publicly positions itself aggressively on price
- Dealerware positions itself in a premium enterprise lane and publicly prices
  connected hardware/toll services much higher
- RENTALL validates a modular pricing model with optional capability add-ons

## Why This Pricing Makes Sense

This pricing is recommended because it:

- gives the client a real founding-client advantage
- keeps the monthly cost approachable
- avoids underpricing implementation and support
- creates a clean path for future pricing increases on later clients
- preserves add-on pricing for higher-value workflows

## Commercial Recommendation

The strongest commercial recommendation is:

- present `Option C` as the full value package
- be ready to close on `Option B`
- keep `Option A` as the lower-friction entry point

If needed, the concession should happen in setup or launch terms, not by
destroying the recurring monthly value.

Recommended concession options:

- keep setup at `$1,500` and include `Customer Service` free for 60 days
- or reduce setup to `$1,000` while keeping the monthly package intact

## Suggested Client-Facing Language

Ride Fleet is pleased to offer founding client pricing for your dealership
loaner operation. For up to 200 vehicles, the platform can be launched with a
one-time setup fee of `$1,500` and a monthly subscription of `$1,250` for the
Ride Fleet core platform with the Loaner Program included. Optional add-ons are
available for Customer Service at `$250/month` and Puerto Rico Tolls at
`$300/month`. This structure gives your team a modern and scalable operating
platform at a preferred early-client rate while keeping room for future
automation and support workflows as the operation grows.

## Final Recommendation

Recommended pricing for this client:

- Setup Fee: `$1,500`
- Core + Loaner Program up to 200 vehicles: `$1,250 / month`
- Customer Service: `$250 / month`
- Tolls: `$300 / month`

Recommended package to lead with:

- `Option C`
- `$1,500` setup
- `$1,800 / month`
