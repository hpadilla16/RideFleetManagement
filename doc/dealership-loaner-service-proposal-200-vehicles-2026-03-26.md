# Ride Fleet Proposal

## Dealership Loaner Program

Fecha: 2026-03-26

## Resumen Ejecutivo

Esta propuesta recomienda una oferta comercial para un cliente dealership que
quiere operar hasta `200` vehículos bajo el `Loaner Program` de Ride Fleet.

La recomendación busca:

- mantenerse competitiva frente al mercado
- no caer en pricing demasiado barato que quite margen
- cobrar de forma clara por valor real
- separar módulos premium que no todos los rooftops necesitan desde el día uno

La estructura propuesta es:

- `one-time setup fee`
- `base monthly platform fee`
- `loaner add-on` incluido en la oferta base
- add-ons opcionales para:
  - `Customer Service`
  - `Tolls`

## Lectura de Mercado

### 1. ARSLoaner

Fuente pública:

- https://arsloaner.com/pricing.aspx
- https://arsloaner.com/features.aspx

Señales importantes:

- publica pricing agresivo
- indica:
  - `$10 / VIN / month`
  - `less for larger fleets`
  - `max $250 per rooftop`
- incluye training
- no cobra setup fee según su positioning público

Lectura:

- ARS compite por precio
- probablemente gana deals sensibles a costo
- pero su ancla de mercado es “simple, affordable, dealership-first”

### 2. Dealerware

Fuentes públicas:

- https://www.dealerware.com/loaner-fleet/
- https://www.dealerware.com/better-loaner-software/
- https://support.dealerware.com/hc/en-us/articles/360022445833-Place-a-Fleet-Increase-or-First-Time-Order-for-Connected-Car-Devices-or-Toll-Equipment

Señales importantes:

- no publica el precio base de plataforma
- se posiciona como premium / enterprise
- habla de:
  - ahorro promedio de `$70` por vehículo por mes vía cost recovery
  - onboarding incluido
  - soporte prioritario incluido
- sí publica add-ons de hardware/telematics:
  - `Connected Car Device Services`: `$20/device/month`
  - `Toll Tag Services`: `$10/VIN/month`

Lectura:

- Dealerware ancla el lado alto del mercado
- tolera tickets mensuales mucho más altos si hay automatización fuerte y revenue recovery
- la referencia de `tolls = $10/VIN/month` es valiosa como benchmark premium

### 3. RENTALL

Fuentes públicas:

- https://www.rentallsoftware.com/pricing/
- https://www.rentallsoftware.com/dealership-loaner/
- https://www.rentallsoftware.com/online-reservation-plugin/

  - `Essential`
  - `Business`
  - `Enterprise`
- modulariza add-ons como:
  - reservation plugin
  - API
  - mobile app
  - payments
- FAQ pública confirma que integran reservation module dentro del website existente
- Help Center confirma que ciertas capacidades se activan como add-on

Lectura:

- RENTALL compite por modularidad y flexibilidad
- es una referencia útil para justificar:
  - base platform
  - paid add-ons
  - website/booking extras

## Conclusión de Mercado

El mercado parece dividirse así:

- `ARSLoaner`: bajo precio
- `Dealerware`: premium / enterprise / recovery-first
- `RENTALL`: modular / custom quote

La mejor posición de Ride Fleet para este deal es:

- más moderno y flexible que ARS
- más accesible que Dealerware
- más transparente y dealership-specific que una propuesta genérica custom

## Recomendación Comercial

### One-Time Setup Fee

Recomendación:

- `$3,500` one-time

Qué incluye:

- tenant setup
- module activation
- user and role setup
- basic workflow configuration
- agreement / forms configuration
- initial fleet import templates
- training for admin + ops team
- go-live support

Justificación:

- suficientemente serio para no regalar implementación
- sigue siendo razonable para un rooftop con hasta 200 unidades
- muy por debajo de lo que normalmente cobraría una implementación enterprise más pesada

### Base Subscription

Recomendación:

- `$1,750 / month`

Qué incluye:

- Ride Fleet core platform
- Loaner Program module
- hasta `200` vehículos activos
- reservations
- customer records
- agreements
- check-out / check-in
- inspections
- payments workflow
- reporting
- employee workflow
- manager visibility
- standard onboarding and support

Equivalencia:

- `~$8.75 por vehículo / mes` a 200 unidades

Lectura competitiva:

- por encima del ultra-low-cost
- claramente por debajo del tipo de economics premium que sugiere Dealerware
- razonable para una operación que quiere capacidad real y room to grow

## Add-Ons Recomendados

### 1. Customer Service Module

Recomendación:

- `$350 / month`

Incluye:

- Issue Center
- case handling
- dispute workflow
- internal support triage
- case history
- request more info / replies

Razón para cobrarlo aparte:

- no todos los rooftops necesitan formalizar customer service desde el inicio
- agrega valor operativo claro para post-rental, escalations y QA

### 2. Tolls Module

Recomendación:

- `$450 / month`

Incluye:

- Puerto Rico AutoExpreso setup
- provider connection
- live sync
- bulk/manual import fallback
- automatic matching against tenant fleet and reservations
- toll review queue
- dispute linkage into Issue Center

Justificación:

- sigue siendo extremadamente competitivo frente a la referencia premium de Dealerware
- Dealerware publica `Toll Tag Services: $10/VIN/month`
- a 200 vehículos eso sería una referencia de `~$2,000 / month`
- Ride Fleet a `$450 / month` queda bien agresivo sin regalar el valor

## Oferta Recomendada Para Cerrar

La recomendación más fuerte para cerrar el deal es:

- presentar `Opción C` como la visión completa
- pero ofrecer `Opción B` o `Opción A` como punto de entrada

Mi recomendación concreta para negociación:

- presentar:
  - setup fee `$3,500`
  - base monthly `$1,750`
  - customer service `$350`
  - tolls `$450`
- y ofrecer una concesión comercial así:
  - `waive 50% of setup fee` o
  - `include Customer Service free for first 60 days`

Eso se ve bien frente al cliente porque:

- no bajas el monthly demasiado rápido
- mantienes el valor de los módulos premium
- das una concesión visible y entendible

## Recomendación Final

Para este cliente yo propondría oficialmente:

- `Setup Fee`: `$3,500`
- `Base Monthly + Loaner Program (up to 200 vehicles)`: `$1,750`
- `Customer Service Module`: `$350 / month`
- `Tolls Module`: `$450 / month`

Y si quieres un close más agresivo:

- `Launch Special`
  - setup fee reducido a `$2,500`
  - o Customer Service incluido por `60 días`
