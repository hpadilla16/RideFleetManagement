# Reservations Date Filter — Quick Guide

**Date:** 2026-04-29
**Page:** `/reservations` (back-office)

---

## English

The "Pickup from" / "To" date filter narrows the reservation list to bookings whose **pickup date** falls within a chosen range. It does NOT filter by return date or by when the reservation was created.

### Common uses

**Today's pickups**

- Set "Pickup from" to today's date.
- Set "To" to today's date.
- List shows only customers picking up today.

**This week's pickups**

- Set "Pickup from" to Monday.
- Set "To" to Sunday.
- List shows all pickups during the week.

**Everything from a date forward**

- Set "Pickup from" to the date.
- Leave "To" empty.
- List shows that date and every pickup after.

**Everything up to a date**

- Leave "Pickup from" empty.
- Set "To" to the date.
- List shows every pickup on or before.

### Important — single-day filter

To filter for **a single day only**, you must set **both** "Pickup from" AND "To" to the same date. Setting only one field creates an open-ended range.

### Clearing the filter

When either date is set, a **"Clear dates"** button appears. Click it to remove all date filtering and see the full list again.

### Combining with search

The text search box and the date filter work together. Example: type `smith` in the search and pick April 29 in both date fields → the list shows only reservations on April 29 where the customer name or reservation number contains "smith".

## Español

El filtro de fechas **"Pickup from" / "To"** limita la lista de reservaciones a las que tienen **fecha de salida (pickup)** dentro del rango elegido. NO filtra por fecha de devolución ni por cuándo se creó la reservación.

### Usos comunes

**Pickups de hoy**

- Pon "Pickup from" en la fecha de hoy.
- Pon "To" en la fecha de hoy.
- La lista muestra solo los clientes que recogen vehículo hoy.

**Pickups de la semana**

- Pon "Pickup from" en el lunes.
- Pon "To" en el domingo.
- La lista muestra todos los pickups de esos días.

**Todos los pickups desde una fecha en adelante**

- Pon "Pickup from" en la fecha deseada.
- Deja "To" en blanco.
- La lista muestra esa fecha y todos los pickups posteriores.

**Todos los pickups hasta una fecha**

- Deja "Pickup from" en blanco.
- Pon "To" en la fecha deseada.
- La lista muestra todos los pickups en o antes de esa fecha.

### Importante — filtro de un solo día

Para filtrar **un solo día**, hay que poner **ambos** campos ("Pickup from" Y "To") en la misma fecha. Si solo pones uno, se crea un rango abierto (sin fecha de cierre).

### Limpiar el filtro

Cuando cualquiera de las dos fechas está puesta, aparece el botón **"Clear dates"**. Haz clic para quitar el filtro completo y ver toda la lista de nuevo.

### Combinar con búsqueda

El campo de búsqueda de texto y el filtro de fechas trabajan juntos. Ejemplo: escribe `smith` en la búsqueda y elige 29 de abril en los dos campos de fecha → la lista solo muestra reservaciones del 29 de abril donde el nombre del cliente o el número de reservación contiene "smith".
